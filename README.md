# Flask CRUD application
A simple dashboard application for tracking progect status assigned to students. Built using Flask, SQLite and Bootstrap

## How to use
Initially, run the <code>database.py</code> program by typing </code>python database.py</code>. This command will create a SQLite database file. Run this file only once.
To start the application, type <code>python app.py</code> and open up a web browser and navigate to <code>http://localhost:5000</code>
